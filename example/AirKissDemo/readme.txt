This demo is working with CC3200SDK 1.1.0.

The steps to compile this demo:
1, Install CC3200SDK 1.1.0 which can be downloaded from www.ti.com.
2, Decompress this zip file into c:\ti\CC3200SDK_1.1.0\cc3200sdk\examples
3, Open c:\ti\CC3200SDK_1.1.0\cc3200sdk\examples\AirKissDemo\ewarm\AirKissDemo.eww with IAR
4, Rebuild the project.

NOTE: If you need to debug it, JTAG is workable, also you can enable serial console with NOTERM parameter disabled in IAR.
